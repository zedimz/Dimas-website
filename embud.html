Proyek Saya

Saya sedang mengikuti kelas Cyber Security dan memiliki proyek di HTML, CSS, JS.

HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat halaman web. Ini digunakan untuk struktur konten web.

CSS (Cascading Style Sheets) digunakan untuk mengatur tampilan dan gaya dari halaman web, seperti warna, font, dan layout.

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web interaktif, seperti animasi dan respons terhadap aksi pengguna.
